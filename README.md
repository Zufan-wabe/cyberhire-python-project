# CyberHire — Cybersecurity Job Application Tracker

## Setup (5 minutes)

1. Install dependencies:
   pip install -r requirements.txt

2. Add your free Anthropic API key:
   cp .env.example .env
   # Edit .env and paste your key from console.anthropic.com

3. Run the app:
   python app.py

4. Open your browser: http://localhost:5000

## Getting a FREE Anthropic API Key
- Go to https://console.anthropic.com
- Sign up for free
- Create an API key
- Paste it in your .env file

## Features
- Track all your cybersecurity job applications
- Dashboard with stats (applied, interviewing, offers, rejected)
- AI-generated follow-up emails with one click
- Add notes, contacts, job URLs for each application
- Update status as you progress
